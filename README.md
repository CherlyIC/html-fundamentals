# Chocolate Cake Guide
## Project Description
This is a multi-page website built as part of the HTML Fundamentals assignment. 
The website is themed around a Chocolate Cake Guide and demonstrates core HTML 
concepts including tables, lists, images, forms, and multi-page navigation.

## What This Project Demonstrates
- Correct HTML page structure and syntax
- Semantic HTML elements
- Ordered and unordered lists
- Image embedding with alt text
- Form creation with labels, radio buttons, checkboxes and a submit button
- Multi-page navigation using anchor tags
- Data tables with caption, thead and tbody
- HTML text formatting tags reference table
- GitHub version control with meaningful commits
